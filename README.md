# RSAS 漏洞扫描任务批量管理工具 (RSAS Manager GUI)

这是一个基于 Python `tkinter` 开发的图形化界面工具，专为绿盟 RSAS (远程安全评估系统) 设计。它允许用户无需登录 Web 界面即可完成登录验证、查看任务列表以及**批量下发 Web 扫描任务**。

主要用于简化渗透测试人员或运维人员的重复性操作，支持通过 HTTP 代理（如 Burp Suite）进行流量调试。

## 🛠 功能特性

  * **图形化界面 (GUI)**: 使用 `tkinter` 构建，采用 Clam 主题，界面整洁美观。
  * **登录会话管理**:
      * 自动处理 CSRF Token。
      * 保持 Session 会话状态。
      * 支持自定义超时时间。
  * **任务列表获取**: 解析 RSAS 系统后台 HTML，以表格形式展示当前任务 ID、风险等级、进度和名称。
  * **Web 任务下发**:
      * **单任务模式**: 指定任务名和 URL 下发。
      * **批量模式**: 支持文本导入，一键批量创建扫描任务。
  * **网络配置**: 支持配置目标系统 URL 及 HTTP/HTTPS 代理。
  * **日志系统**: 界面内置彩色日志窗口，实时反馈操作成功 (绿色)、失败 (红色) 或处理中 (蓝色) 的状态。

## 📋 环境依赖

请确保你的 Python 环境安装了以下库：

  * Python 3.x
  * `tkinter` (通常随 Python 安装)
  * `requests`
  * `beautifulsoup4`

安装命令：

```bash
pip install requests beautifulsoup4
```

## 🚀 快速开始

1.  **运行脚本**:

    ```bash
    python rsas_manager.py
    ```

2.  **配置连接**:

      * **系统地址**: 输入 RSAS 的登录地址 (例如 `https://192.168.0.7`)。
      * **代理地址**: (可选) 如果需要抓包调试，勾选“启用”并填写代理地址 (如 `http://127.0.0.1:8083`)。
      * **账号密码**: 输入 RSAS 管理员账号。

3.  **操作流程**:

      * 点击 **"1. 点击登录"**。观察日志区域，显示绿色 "已登录" 即为成功。
      * 登录成功后，点击 **"2. 获取任务列表"** 可刷新右侧表格查看现有任务。

## 📂 批量任务下发说明

在 **"任务下发"** -\> **"自定义批量下发"** 标签页中，支持批量导入目标。

### 格式要求

  * 每行一条记录。
  * 使用 **英文分号 (;)** 分隔任务名称和目标 URL。
  * 格式：`任务名称; 目标URL`

### 示例数据

将以下内容复制到输入框中：

```text
电商平台前台扫描; http://shop.example.com
ERP系统扫描; http://erp.internal.com:8080
API接口测试; https://api.test-site.org
```

点击 **"解析并批量下发"** 按钮，工具将自动逐个发送请求，并添加延时防止并发过高。
